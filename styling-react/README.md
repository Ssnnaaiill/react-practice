## Sass
Sass는 CSS pre-processer로서, 다음과 같은 특징이 있습니다.
1. 복잡한 작업을 쉽게 할 수 있게 합니다.
2. 코드의 재활용성을 높여줍니다.
3. 코드의 가독성을 높여 유지보수가 쉬워집니다.


### 준비
Sass를 CSS로 변화시켜주는 `node-sass` 모듈을 설치합니다.
```
$ yarn add node-sass
```
